ARTiCUT APPLICATION

DESCRIPTION

The ARTiCUT Application allows users to digitally design Pabalat patterns using grid-based tools, drawing features, and precision controls before sending designs for automated cutting.

ARTiCUT DEVICE

DESCRIPTION

The ARTiCUT Device is a precision cutting machine designed to translate digital Pabalat designs into physical artwork, preserving intricate details through controlled motion and accurate blade placement .
